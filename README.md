## Features

- Create an account on chatbot.tradomate.io
- Create AI Chatbots
- Deploy chatbots to chatbot.tradomate.io

## Requirements

- Python 3.6+
- Click librarymn
- Requests library

## Installation

1. Clone this repository:

   ```bash
   git clone git
   ```

2. Install the requirements:

   ```bash
   pip install -r requirements.txt
   ```

3. Install the CLI:

   ```bash
   pip install .
   ```

4. Run the CLI:

   ```bash
   tradomate-cli
   ```
